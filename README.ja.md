# mizuyokan

福井県にある「水ようかん」（伝統的な日本の冬の和菓子）の店舗を表示するWebマップです。このプロジェクトでは、2011年の店舗データを可視化しています。

## デモ

**ライブアプリケーション: [https://github.com/code4fukui/mizuyokan](https://github.com/code4fukui/mizuyokan)**

ユーザーインターフェースには、店舗の位置を示すピンが配置されたインタラクティブなマップが表示されます。ピンをクリックすると、店舗名、住所、電話番号、価格、水ようかんの写真がポップアップで表示されます。マップと並んで、フィルタリングや並べ替えが可能な全店舗データのテーブルも表示されます。

## 機能

- **インタラクティブマップ**: データセットに含まれるすべての水ようかん店舗の位置を表示します。
- **詳細情報**: 各マップマーカーは、写真を含む店舗の詳細情報を提供します。
- **データテーブル**: `<csv-viewer>`コンポーネントにより、生データを使いやすいテーブル形式で表示します。
- **静的サイト**: サーバーサイドのバックエンドを必要とせず、ブラウザだけで動作します。
- **レガシーバージョン**: Google Maps APIを使用した古い実装（`index-old-googlemaps-ver.html`）も含まれています。

## 仕組み

このプロジェクトは、モダンなWebコンポーネントを使用してデータを表示する静的なHTMLページです:

- `<csv-map>`: `fukui-mizuyokan.csv`ファイルからインタラクティブマップとマーカーを直接レンダリングします。
- `<csv-viewer>`: 同じCSVファイルからデータテーブルをレンダリングします。

これらのコンポーネントはそれぞれのリポジトリからインポートされるため、ローカルでのインストールやビルドステップは必要ありません。

## データ

店舗情報は [`fukui-mizuyokan.csv`](fukui-mizuyokan.csv) に保存されています。データは2011年のもので、以下の列を含みます:

- `店舗名` (Shop Name)
- `電話番号` (Phone Number)
- `住所` (Address)
- `緯度` (Latitude)
- `経度` (Longitude)
- `写真URL` (Photo URL)
- `価格` (Price)

## 使い方

このプロジェクトをローカルで実行するには、`index.html`ファイルをWebブラウザで開くだけです。
